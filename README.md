# 🎬 CineIsadora – Meu Site de Cinema

Projeto criado para a disciplina de Desenvolvimento Web. Um site de cinema com meus filmes favoritos, onde os visitantes podem explorar títulos, favoritar filmes e entrar em contato.

## 🔗 Links

- **Site:** [Clique aqui para acessar](https://SEU-USUARIO.github.io/meu-site-de-cinema/)
- **Repositório:** [github.com/SEU-USUARIO/meu-site-de-cinema](https://github.com/SEU-USUARIO/meu-site-de-cinema)

## 📋 Funcionalidades

- 🎥 Vitrine com **15 filmes favoritos** organizados em 3 categorias
- 🔍 **Busca em tempo real** por nome do filme
- 🗂️ **Filtros por categoria** (Clássicos, Ação & Aventura, Drama & Suspense)
- ❤️ **Sistema de favoritos** salvo no navegador
- 📩 **Página de contato** com formulário (Nome, E-mail e Mensagem)
- 📱 Design **responsivo** para celular e computador

## 🛠️ Tecnologias Utilizadas

- **HTML5** — estrutura das páginas
- **CSS3** — estilização com Flexbox e Grid
- **JavaScript** — interatividade (busca, filtros, favoritos, modal)

## 📁 Estrutura do Projeto

```
📦 meu-site-de-cinema
 ┣ 📄 index.html      → Página principal com os filmes
 ┣ 📄 contato.html    → Página de contato
 ┣ 🎨 style.css       → Estilos e design
 ┗ ⚙️ script.js       → Funções JavaScript
```
